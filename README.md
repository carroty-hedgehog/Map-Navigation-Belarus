# Map-Navigation-Belarus
Интерактивный цифровой атлас-навигатор по логистическим и туристическим объектам Беларуси

версия 1
необходимы правки:
1. "distance": и "time": - при клике должны исчезать вместе с соответствующей линией
2. локации на карте отображаются даже, если кнопка локации не активна. Локации всех уровней должны пропадать, если кнопка "Локации" не активирована

версия 2
необходимы правки:
1. Линии нельзя повторно отобразить/скрыть после активации кнопки локации
2. Названия локаций отображаются только, если на них нажать
3. При наведении на название города курсор меняется что б тыкнуть, хотя поле не кликабельное
4. Некоторые локации некорректно отображаются на карте
